# MindMaps
1. some mindmap for erlang app such as ranch and a game app

## 说明
1. ranch_app是erlang的一个开源tcp服务应用。ranch_app.xmind是研究这个框架时所作的思维导图，可以帮助读者更清晰的把握整个应用的纹理架构。后续有完善的地方会及时上传。因本人水平有限，若有不足之处请不吝赐教。
2. yxyz.xmind是页游英雄远征的整个游戏框架的思维导图。可帮助理解这个游戏的框架。

## 需要工具
1. 因这几个思维导图是用xmind画的，所以查看也需要xmind来实现。xmind的下载可到网上搜寻。
